## บทคัดย่อ

โครงงานนี้นำเสนอการออกแบบและพัฒนา **ระบบควบคุมหลอดไฟอัจฉริยะ** ที่ทำงานบนเครือข่ายภายในบ้าน (LAN)
โดยใช้โพรโทคอล **MQTT** เป็นช่องทางสื่อสารระหว่างอุปกรณ์ฮาร์ดแวร์กับเซิร์ฟเวอร์ จุดเด่นของระบบคือการ
ออกแบบให้ **เซิร์ฟเวอร์ (Backend) เป็นศูนย์กลางความจริงเพียงหนึ่งเดียว (Single Source of Truth)**
กล่าวคือ ทุกการเปลี่ยนแปลงสถานะของหลอดไฟ ไม่ว่าจะมาจากการกดปุ่มบนหน้าเว็บ การสั่งงานด้วยเสียง
การตั้งเวลาอัตโนมัติ หรือสัญญาณจากตัวอุปกรณ์เอง จะถูกบันทึกและตัดสินที่เซิร์ฟเวอร์เป็นจุดเดียว
แล้วกระจาย (broadcast) สถานะล่าสุดไปยังหน้าเว็บทุกเครื่องแบบเรียลไทม์ผ่าน **Server-Sent Events (SSE)**

ระบบประกอบด้วยสามส่วนหลัก ได้แก่ (1) เฟิร์มแวร์บนไมโครคอนโทรลเลอร์ **ESP8266 (NodeMCU)** ที่ควบคุม
รีเลย์เปิด/ปิดไฟ พร้อมรองรับการสั่งงานด้วยเสียงผ่านโมดูล VC-02 (2) เซิร์ฟเวอร์ที่พัฒนาด้วย **Express 5
และ MongoDB** ทำหน้าที่เป็นทั้งผู้รับ-ส่งข้อความ MQTT, REST API และต้นทาง SSE และ (3) หน้าเว็บที่พัฒนา
ด้วย **Vue 3** สำหรับแสดงสถานะและสั่งงาน

ระบบได้ออกแบบกลไกสำคัญเพื่อให้มีความน่าเชื่อถือ ได้แก่ การจับคู่อุปกรณ์อัตโนมัติแบบ **Announce & Claim**
ที่ผู้ใช้ไม่ต้องตั้งรหัสอุปกรณ์เอง การตรวจจับอุปกรณ์ออฟไลน์จาก 3 ช่องทาง และกลไกปรับสถานะให้ตรงกัน
(drift reconciliation) ระหว่างสถานะในฐานข้อมูลกับฮาร์ดแวร์จริงผ่านสัญญาณ heartbeat
